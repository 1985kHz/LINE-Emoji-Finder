## Background

When looking for a LINE emoji to accompany a message, there is no way to search by the text displayed within the emoji image itself. This tool was built to solve that problem.

## Overview

A personal web service that searches and displays LINE emojis matching user-entered text.

OCR is applied to several hundred LINE emojis published by creators the author frequently uses, and the recognition results are stored as a JSON file. The app queries that JSON to find emojis containing the input text.
